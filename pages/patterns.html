What is a Mathematical Pattern?

A pattern is a sequence of numbers or objects that follows a predictable rule. Recognising patterns is a fundamental skill in mathematics — from simple counting to the algorithms that power search engines and AI.

Every pattern has a general term formula (the nth term), which lets you calculate any term directly without working through every prior term. Click a card below to open its generator.

Pattern Generators

Arithmetic Pattern (AP)

Add a constant difference d each time. Example: 3, 7, 11, 15…
aₙ = a + (n−1)d
+
d can be positive (growing) or negative (shrinking). Applications include uniform motion (equal distances in equal times), simple interest, and evenly spaced design grids.

Geometric Pattern (GP)

Multiply by constant ratio r each time. Example: 2, 6, 18, 54…
aₙ = a × r^(n−1)
+
|r| > 1 → exponential growth; |r| < 1 → decay toward zero. Models compound interest, population growth, radioactive decay and computer storage (KB → MB → GB, r = 1024).

Square Numbers

n × n for each positive integer. Example: 1, 4, 9, 16, 25…
aₙ = n²
+
The gap between consecutive squares is always an odd number — because n² − (n−1)² = 2n−1. Square numbers are central to Pythagoras' theorem and quadratic equations.

Triangular Numbers

Dots forming equilateral triangles. Example: 1, 3, 6, 10, 15…
Tₙ = n(n+1) / 2
+
Tₙ = 1+2+3+…+n — the sum Gauss computed in seconds as a child. They appear in Pascal's Triangle, in combinations C(n+1,2) = Tₙ, and in handshake-counting problems.

Fibonacci Sequence

Each term is the sum of the previous two. 0, 1, 1, 2, 3, 5, 8, 13…
Fₙ = Fₙ₋₁ + Fₙ₋₂
+
As n→∞, Fₙ/Fₙ₋₁ → φ = (1+√5)/2 ≈ 1.618 (the Golden Ratio). Appears in sunflower spirals, nautilus shells, pinecones, galaxy arms and classical architecture proportions.

Star / Dot Pattern

A visual triangle — row n contains exactly n stars.
Row n → n stars
+
Total stars after n rows = 1+2+…+n = Tₙ (triangular number). The classic introduction to nested loops in programming; every row adds one more star than the last.
