Patterns are the heartbeat of mathematics. Discover the rules behind famous sequences and generate them interactively.

What is a Mathematical Pattern?

A pattern is a sequence following a predictable rule. Recognising patterns is the core of mathematical thinking — from elementary counting to machine learning. Patterns appear in flower petals (Fibonacci), crystal structures (geometric), and financial models (arithmetic).

Every pattern has a general term formula (nth term) that lets you jump to any position without computing all prior terms. Click any card below to open its generator.

Pattern Generators

Arithmetic Pattern (AP)

Each term changes by a constant difference d. Example: 3, 7, 11, 15…
aₙ = a + (n−1)d
+
The common difference d can be positive (growing) or negative (shrinking). Applications: uniform motion distances, simple interest, salary increments. Max 50 terms generated.

Geometric Pattern (GP)

Each term is multiplied by a constant ratio r. Example: 2, 6, 18, 54…
aₙ = a × r^(n−1)
+
When |r| > 1 the sequence grows exponentially; when |r| < 1 it decays toward zero. Models: population growth, compound interest, radioactive decay, and computer memory (KB → MB → GB, r = 1024).

Square Numbers

Product of an integer with itself. Example: 1, 4, 9, 16, 25…
aₙ = n²
+
The gap between consecutive squares is always an odd number (3, 5, 7…) because n² − (n−1)² = 2n−1. Square numbers are central to the Pythagorean theorem and quadratic equations.

Triangular Numbers

Dots arranged in equilateral triangles. Example: 1, 3, 6, 10, 15…
Tₙ = n(n+1) / 2
+
Tₙ = 1+2+3+…+n — the formula Gauss used to sum 1–100 instantly. They appear in Pascal's Triangle (2nd diagonal), combinatorics C(n+1,2) = Tₙ, and handshake counting problems.

Fibonacci Sequence

Each term = sum of previous two. Example: 0, 1, 1, 2, 3, 5, 8, 13…
Fₙ = Fₙ₋₁ + Fₙ₋₂
+
As n → ∞, Fₙ/Fₙ₋₁ → φ = (1+√5)/2 ≈ 1.618 (the Golden Ratio). Found in sunflower spirals, nautilus shells, galaxy arms, and the proportions of classical architecture.

Star / Dot Pattern

Visual triangle of stars — row n has exactly n stars. Classic loop exercise.
Stars in row n = n
+
Total stars after n rows = 1+2+…+n = Tₙ (triangular number). This pattern is a classic introduction to nested loops in programming and visualises arithmetic series row by row.
