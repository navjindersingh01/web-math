# Web Math

## Overview

Web Math is a multi-page educational web application designed to help users understand fundamental mathematical concepts through structured content and interactive tools. The platform focuses on three key areas: geometric areas, mathematical patterns, and numerical series.

The application combines conceptual explanations with interactive calculators and generators to enhance learning and usability.

---

## Features

### 1. Areas

* Covers 2D and 3D shapes including:

  * Circle, Rectangle, Triangle, Square
  * Parallelogram, Trapezium, Ellipse, Sphere
* Each shape includes:

  * Visual representation
  * Parameter labeling
  * Formula explanation
  * Interactive calculator for computing area or surface area

---

### 2. Patterns

* Includes multiple pattern types:

  * Arithmetic Pattern
  * Geometric Pattern
  * Square Pattern
  * Triangular Numbers
  * Fibonacci Pattern
  * Dot Pattern
* Features:

  * Rule-based explanation
  * Pattern generator based on user input
  * Output display with copy and reset functionality

---

### 3. Series

* Covers:

  * Arithmetic Series (AP)
  * Geometric Series (GP)
  * Infinite Geometric Series
  * Harmonic Series
* Each section includes:

  * Formula and concept explanation
  * Interactive calculator
  * Step-based interpretation of results

---

### 4. Author Page

* Displays author information including:

  * Profile image
  * Educational background
  * Professional summary
  * Links to GitHub and LinkedIn
